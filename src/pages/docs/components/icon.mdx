    <Section title="Icon System">
        ### Description
        Icons are implemented as React components that inherit `currentColor` for styling, making them easily themeable and customizable. They default to a medium size (5x5 units) but can be resized using Tailwind's `size` utilities.

        ### Usage
        ```tsx
        import { IconSearch, IconChevronDown, IconLocation } from "crafted-ui";

        <IconSearch />
        <IconChevronDown className="size-6" />
        <IconLocation className="text-primary" />
        ```

        ### Props
        All icon components accept the following props:
        
        | Prop | Type | Default | Description |
        | --- | --- | --- | --- |
        | className | string | - | Additional CSS class names for styling |
        | Any SVG prop | - | - | Standard SVG attributes (stroke, fill, etc.) |

        Icons use `currentColor` by default, meaning they inherit the text color from their parent element.
    </Section>
