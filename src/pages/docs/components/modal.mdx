<Section title="Overview">
    The Modal component provides a dialog interface for displaying content that overlays the main application. It's built on Base UI's Dialog primitive, offering full accessibility support with proper focus management, keyboard navigation, and ARIA attributes.
    
    The Modal component is composable, consisting of several sub-components that work together to create complete dialog experiences.
</Section>

<Section title="Components">
    ### Modal
    
    The root component that manages the modal state and wraps all other modal components.
    
    **Usage:**
    ```tsx
    import { Modal } from "crafted-ui";
    
    <Modal>
        {/* Modal sub-components */}
    </Modal>
    ```
    
    **Props:**
    
    | Prop | Type | Default | Description |
    | --- | --- | --- | --- |
    | open | boolean | - | Controlled open state |
    | defaultOpen | boolean | false | Uncontrolled default open state |
    | onOpenChange | (open: boolean) => void | - | Callback when open state changes |
    | className | string | - | Additional CSS class names |
    
    Accepts all props from the Base UI Dialog.Root component.
    
    ### ModalTrigger
    
    A button that opens the modal when clicked.
    
    **Usage:**
    ```tsx
    import { ModalTrigger } from "crafted-ui";
    
    <ModalTrigger>
        <Button>Open Modal</Button>
    </ModalTrigger>
    ```
    
    **Props:**
    
    | Prop | Type | Default | Description |
    | --- | --- | --- | --- |
    | className | string | - | Additional CSS class names |
    | children | ReactNode | - | Content of the trigger (typically a Button) |
    
    Accepts all props from the Base UI Dialog.Trigger component.
    
    ### ModalContent
    
    The main modal dialog container that displays the modal content with optional title and subtitle.
    
    **Usage:**
    ```tsx
    import { ModalContent } from "crafted-ui";
    
    <ModalContent title="Modal Title" subtitle="Optional subtitle">
        {/* Modal content */}
    </ModalContent>
    ```
    
    **Props:**
    
    | Prop | Type | Default | Description |
    | --- | --- | --- | --- |
    | title | string | - | Modal title (automatically includes close button) |
    | subtitle | string | - | Optional subtitle text |
    | size | "sm" \| "md" \| "lg" \| "xl" | "md" | Width of the modal dialog |
    | className | string | - | Additional CSS class names |
    | children | ReactNode | - | Content of the modal |
    
    Accepts all props from the Base UI Dialog.Popup component.
    
    ### ModalTitle
    
    Component for rendering the modal title with optional close button. Usually not needed if you pass `title` prop to ModalContent.
    
    **Props:**
    
    | Prop | Type | Default | Description |
    | --- | --- | --- | --- |
    | title | string | - | Title text |
    | subtitle | string | - | Optional subtitle text |
    | className | string | - | Additional CSS class names |
    
    ### ModalDescription
    
    Descriptive text for the modal content with accessible styling.
    
    **Usage:**
    ```tsx
    import { ModalDescription } from "crafted-ui";
    
    <ModalDescription>
        This modal contains important information...
    </ModalDescription>
    ```
    
    **Props:**
    
    | Prop | Type | Default | Description |
    | --- | --- | --- | --- |
    | className | string | - | Additional CSS class names |
    | children | ReactNode | - | Description text |
    
    ### ModalClose
    
    A button that closes the modal when clicked. Wrap around any button or interactive element.
    
    **Usage:**
    ```tsx
    import { ModalClose } from "crafted-ui";
    
    <ModalClose>
        <Button>Cancel</Button>
    </ModalClose>
    ```
    
    **Props:**
    
    | Prop | Type | Default | Description |
    | --- | --- | --- | --- |
    | className | string | - | Additional CSS class names |
    | children | ReactNode | - | Content (typically a Button) |
</Section>

<Section title="Modal Sizes">
    The Modal supports four size options that control the width of the dialog:
    
    - **sm** (384px) - Compact content, simple confirmations
    - **md** (512px) - Forms with 1-2 columns (default)
    - **lg** (640px) - Wider forms or data tables
    - **xl** (768px) - Complex layouts with multiple sections
    
    All modals are responsive and have a maximum width of `calc(100vw - 3rem)` to ensure proper display on smaller screens.
</Section>

<Section title="Accessibility">
    The Modal component is built on Base UI's Dialog primitive and includes comprehensive accessibility features:
    
    - **Focus Management** - Focus is trapped within the modal while open and returned to the trigger when closed
    - **Keyboard Navigation** - `Escape` key closes the modal, `Tab` cycles through focusable elements
    - **ARIA Attributes** - Proper `role`, `aria-labelledby`, and `aria-describedby` attributes
    - **Screen Reader Support** - Announces modal state and content appropriately
    - **Backdrop Click** - Clicking outside the modal closes it (customizable)
</Section>
