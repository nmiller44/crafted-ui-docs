    <Section title="Field">
        ### Description
        The root component that wraps the form field elements. It can automatically include a label via the `label` prop.

        ### Usage
    </Section>

    <Section>
        ### Props
        The `Field` component accepts all props from the Base UI Field.Root component, plus:
        
        | Prop | Type | Default | Description |
        | --- | --- | --- | --- |
        | label | string \| ReactNode | - | Label content (automatically renders a FieldLabel component) |
        | className | string | - | Additional CSS class names |
        | children | ReactNode | - | Content of the field |
    </Section>

    <Section title="FieldLabel">
        ### Description
        A label component for the form field.

        ### Props
        The `FieldLabel` component accepts all props from the Base UI Field.Label component, plus:
        
        | Prop | Type | Default | Description |
        | --- | --- | --- | --- |
        | className | string | - | Additional CSS class names |
        | children | ReactNode | - | Content of the label |
    </Section>

    <Section title="FieldError">
        ### Description
        Displays validation error messages. This is automatically included in the Field component and will show errors based on field validation state.

        ### Props
        The `FieldError` component accepts all props from the Base UI Field.Error component, plus:
        
        | Prop | Type | Default | Description |
        | --- | --- | --- | --- |
        | className | string | - | Additional CSS class names |
        | children | ReactNode | - | Custom error content |
    </Section>

    <Section title="FieldInset">
        ### Description
        Adds inset content to the input field, such as units, currency symbols, or icons. The component can display text via the `text` prop or custom content via children.

        ### Basic Usage
    </Section>

    <Section>
        ### Props
        | Prop | Type | Default | Description |
        | --- | --- | --- | --- |
        | text | string | - | Text to display in the inset |
        | position | "left" \| "right" | "right" | Position of the inset |
        | className | string | - | Additional CSS class names |
        | children | ReactNode | - | Custom content to display in the inset |

        ### Usage Notes
        - Input automatically detects FieldInset and adds padding for standard single-character insets (`$`, `%`, etc.)
        - For larger insets, override automatic padding using the `!` (important) modifier:
          - Larger left insets: `className="!pl-16"` or `className="!pl-20"`
          - Larger right insets: `className="!pr-10"` or `className="!pr-12"`
        - Use the `text` prop for simple text insets
        - Use children for custom content like icons or complex elements
    </Section>
