    <Section title="Installation">
        You can install Crafted UI via npm.

        ```bash
        npm install crafted-ui
        ```
    </Section>

    <Section title="Import Styles">
        Import the Crafted UI styles in your application's main CSS file:

        ```css
        @import "tailwindcss";
        @import "crafted-ui/styles" layer(components);
        ```

        This will include all the necessary CSS for the Crafted UI components. **NOTE:** The `layer(components)` directive ensures that Crafted UI styles are properly layered, allowing your custom utility classes to override component defaults.
    </Section>
