        <a href="https://tailwindcss.com/docs/installation" target="_blank" rel="noopener">official installation guide</a>.
    </Section>

    <Section title="Installation">
        You can install Crafted UI via npm.

        ```bash
        npm install crafted-ui
        ```
    </Section>

    <Section title="Import Styles">
        Import the Crafted UI styles in your application's main CSS file:

        ```css
        @import 'tailwindcss';
        @import 'crafted-ui/styles';
        ```

        This will include all the necessary CSS for the Crafted UI components.
    </Section>

    <Section title="Use Crafted UI">
        Import and use Crafted UI components in your React application:
    </Section>
